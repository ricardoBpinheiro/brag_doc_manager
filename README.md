# Brag Doc Manager

## Visão Geral

O Brag Doc Manager é uma aplicação Flutter desenvolvida para ajudar você a gerenciar e acompanhar suas conquistas, feitos e momentos importantes. Ele oferece uma interface intuitiva para criar, visualizar e organizar seus diarios de forma eficiente.

## Funcionalidades Principais

*   **Timeline Dinâmica:** Visualize todas as suas entradas em uma linha do tempo organizada, permitindo uma visão rápida e cronológica de seus feitos.
*   **Editor de Entradas Abrangente:** Crie novas entradas ou edite as existentes com facilidade, adicionando todos os detalhes relevantes de suas conquistas.
*   **Visualização Detalhada:** Acesse uma página de detalhes para cada entrada, onde você pode revisar todas as informações de um feito específico.
*   **Gerenciamento de Dados:** Estrutura robusta para manipulação de dados, garantindo que suas informações sejam armazenadas e recuperadas de forma segura.

## Tecnologias e Arquitetura

Este projeto é construído com **Flutter**, garantindo uma experiência de usuário rica e performática em múltiplas plataformas.

*   **Gerenciamento de Estado:** Utiliza o padrão **BLoC (Business Logic Component)** para um gerenciamento de estado escalável e reativo, separando a lógica de negócios da interface do usuário.
*   **Integração com Firebase:** Prepara a aplicação para utilizar serviços de backend do Firebase, como armazenamento de dados e autenticação, garantindo persistência e sincronização.
*   **Injeção de Dependências:** Implementa um sistema de serviço locator para gerenciar as dependências da aplicação de forma eficiente.
*   **Navegação:** Roteamento de tela gerenciado para uma navegação fluida e consistente.
